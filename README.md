# 🏴‍☠️ Undangan Pernikahan Tema One Piece

Undangan pernikahan digital dengan tema One Piece yang epik! Dengan design Wanted Poster, efek petualangan, dan semua fitur lengkap.

## ⚓ Fitur

- 🎯 **Tema One Piece Autentik** - Wanted poster, bounty, devil fruit effects
- 🎵 **Background Music** - Dari Archive.org
- 📱 **Responsive** - Tampilan optimal di semua device
- 🖼️ **Galeri Foto** - Support IBB.co
- 💰 **Kirim Hadiah** - QRIS & Transfer Bank
- ✅ **RSVP WhatsApp** - Konfirmasi kehadiran
- ⏰ **Countdown** - Menuju hari H
- 🗺️ **Google Maps** - Navigasi lokasi
- 💧 **Watermark** - Design by Malik

## 🚀 Cara Setup

### 1. Edit Config
Buka `config.js` dan ganti semua data di `WEDDING_CONFIG`:
- Data mempelai (nama, foto, bounty, devil fruit)
- Detail acara
- Link IBB untuk galeri & QRIS
- Nomor WA untuk RSVP
- Link musik dari Archive.org
